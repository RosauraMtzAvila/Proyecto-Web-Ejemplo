# Proyecto-Web-Ejemplo

Este es un proyecto de ejemplo de una página web sencilla utilizando HTML y CSS.

## Cómo clonar el proyecto y ejecutar el código en un entorno local

1. Clona el repositorio en tu máquina local:
   ```
   git clone <URL-del-repositorio>
   ```

2. Navega al directorio del proyecto:
   ```
   cd Proyecto-Web-Ejemplo
   ```

3. Abre el archivo `index.html` en tu navegador web preferido para ver la página.

## Ramas del proyecto

- **main**: Rama principal que contiene la versión estable del proyecto. Incluye los archivos básicos `index.html` y `styles.css` para una página web sencilla con un título y un párrafo estilizado.
- **feature-mejora-estilo**: Rama que realiza cambios para mejorar el estilo y la presentación visual del proyecto web.
